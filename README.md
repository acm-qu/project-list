### Statuses
- DONE & MAINTAINED: The project is either complete or almost complete and ready to be shared with the public
- UNDER DEVELOPMENT: The main project lead is assigned and it is under development
- AWAITING DEVELOPER: Only the idea is proposed, no active developer is assigned for the project
